# Calendar Generator in C

A simple calendar generator program developed in C as part of my first-semester **Programming and Problem Solving** course project in 2022.

This was my first programming project, where I practiced basic programming concepts such as input/output, functions, arrays, loops, conditional statements, and leap year calculation.

---

## Project Overview

The Calendar Generator in C takes a year as input from the user and displays the full calendar for that year. It prints all twelve months with proper day alignment and also handles leap years by adjusting the number of days in February.

This project helped me understand how programming logic can be used to solve real-world problems such as date calculation and calendar formatting.

---

## Features

* Takes a year as user input
* Displays all 12 months of the selected year
* Calculates the first day of the year
* Handles leap years
* Uses arrays to store month names and number of days
* Prints the calendar in a structured format
* Demonstrates beginner-level C programming concepts

## How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/calendar-generator-c.git
cd calendar-generator-c
```

### 2. Compile the program

```bash
gcc src/calendar.c -o calendar
```

For Windows PowerShell:

```powershell
gcc src\calendar.c -o calendar.exe
```

### 3. Run the program

For Linux/macOS:

```bash
./calendar
```

For Windows PowerShell:

```powershell
.\calendar.exe
```

### 4. Enter a year

Example:

```text
Enter your searched Year: 2024
```

The program will display the full calendar for the selected year.
